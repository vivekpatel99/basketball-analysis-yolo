## 📊 Format Comparison

### XYXY Format

```
(x1, y1, x2, y2) = (100, 150, 200, 300)

┌─────────────────────────────────┐
│                                 │
│  (100,150) ───────── (200,150)  │
│     │                   │       │
│     │                   │       │
│     │                   │       │
│     │                   │       │
│  (100,300) ───────── (200,300)  │
│                                 │
└─────────────────────────────────┘
```

### XYWH Format

```
(x, y, w, h) = (100, 150, 100, 150)

┌─────────────────────────────────┐
│                                 │
│  (100,150) ───────── (200,150)  │
│     │                   │       │
│     │     w=100         │       │
│     │                   │       │
│     │                   │       │
│  (100,300) ───────── (200,300)  │
│     ↑                   ↑       │
│     h=150               │       │
│                         │       │
└─────────────────────────────────┘
```

### CXCYWH Format

```
(cx, cy, w, h) = (150, 225, 100, 150)

┌─────────────────────────────────┐
│                                 │
│  (100,150) ───────── (200,150)  │
│     │                   │       │
│     │     w=100         │       │
│     │                   │       │
│     │        ● (150,225)│       │ ← Center
│  (100,300) ───────── (200,300)  │
│     ↑                   ↑       │
│     h=150               │       │
│                         │       │
└─────────────────────────────────┘
```

## ⚠️ Common Mistakes Visualization

### Wrong Coordinate Order

```
❌ WRONG: (200, 300, 100, 150)
- x1 > x2 and y1 > y2
- This creates an invalid bbox!

┌─────────────────────────────────┐
│                                 │
│  (100,150) ───────── (200,150)  │
│     │                   │       │
│     │                   │       │
│     │                   │       │
│     │                   │       │
│  (100,300) ───────── (200,300)  │
│                                 │
└─────────────────────────────────┘

✅ CORRECT: (100, 150, 200, 300)
- x1 < x2 and y1 < y2
- Valid bbox!
```

### Y-Axis Confusion

```
❌ WRONG THINKING: Y goes up like in math
┌─────────────────────────────────┐
│                                 │
│  (0,0)                          │
│  │                              │
│  │ Y ↑                          │
│  │                              │
│  │                              │
│  │                              │
│  └──────────────────────────────┘
│  X →

✅ CORRECT: Y goes down in computer vision
┌─────────────────────────────────┐
│ (0,0)                           │
│ │                               │
│ │ Y ↓                           │
│ │                               │
│ │                               │
│ │                               │
│ └───────────────────────────────┘
│ X →
```
